# TapTaxi → Grand Transfer: Интеграция через Telegram

## Идея

Патченый TapTaxi APK автоматически принимает заказы и передаёт их в систему Grand Transfer, где бот раздаёт их водителям.

---

## Как работает

```
[TapTaxi сервер]
      ↓ заказ (откуда, куда, цена)
[Патченый APK на телефоне]
      ↓ автоприём + HTTP запрос
[Grand Transfer API /api/taptaxi-order]
      ↓ считаем цену по нашим тарифам
[Telegram бот]
      ↓ рассылает всем водителям
[Водители Grand Transfer]
      → первый нажавший «✅ Забрать» — берёт заказ
```

---

## Режимы работы APK

### Режим 1 — Полный автомат
APK берёт **все** входящие заказы без участия водителя.
- Скорость: ~50-100мс (быстрее любого человека)
- Минус: берёт всё подряд

### Режим 2 — Фильтр по маршруту
APK берёт только заказы, где пункт назначения содержит нужное слово.
```json
// Конфиг на сервере (не нужна пересборка APK)
{
  "allowed_routes": ["Казань", "Пермь", "Уфа", "Екатеринбург"]
}
```
- Маршрутов можно сколько угодно
- Добавляются/убираются без пересборки APK

### Режим 3 — Полуавтомат через Telegram
Бот присылает диспетчеру уведомление с кнопками:
```
🚖 Новый заказ из TapTaxi
Откуда: Ижевск, ул. Ленина 5
Куда: Казань (420 км)
Цена TapTaxi: 850₽ | Наша цена: 11 760₽

[✅ Взять в работу] [❌ Пропустить]
```

---

## Расчёт цены

Grand Transfer уже считает цену по формуле:
```
Цена = расстояние_км × тариф_за_км
```
Тарифы:
- Econom: 28₽/км
- Standart: 30₽/км
- Comfort: 35₽/км
- Comfort+: 40₽/км
- Minivan: 50₽/км
- Business: 80₽/км

Расстояние берётся из адресов через Yandex Maps API (уже интегрирован).

## Важная особенность

TapTaxi — **агрегатор** с конкурентным принятием заказов.
Наш автоприём работает быстрее людей (~50мс vs 1-3сек).
Это даёт преимущество в гонке за заказ.

После автоприёма в TapTaxi — заказ уже наш, дальше раздаём своим водителям по ценам Grand Transfer.
